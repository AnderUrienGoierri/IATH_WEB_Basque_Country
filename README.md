# Internationalisation at Home

Welcome to the **Internationalisation at Home** project.

This project aims to bring global perspectives to our local community, fostering cultural exchange and international understanding right from home.

## Features

- Welcoming homepage
- Simple and clean design

## About this Repository

This repository contains the source code for the "Internationalisation at Home" web project. It serves as a platform to demonstrate basic web development practices and version control using Git.
